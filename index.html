Welcome!

Lemonade is a task management app that helps you get things done. Learn how Lemonade can help you stay on track with any goal you set for yourself.

Create a task

You can add notes and subtasks for additional context, set a due date, and label your task with tags for easy organization.

Create a new list

Open the main menu to access all your lists, including:

  • Inbox - a place to store general todo items
  • Today - where you can view all your tasks for today
  • Upcoming - which compiles all your tasks for the week

You'll also find the button for creating new ones.

Click Create List to build your first list!

Edit multiple tasks at once

Toggle the button to view additional actions, like:

  • Edit List - to rename your list, or to assign it to a different folder
  • Delete List - to delete one specific list
  • Clear All - to delete all the tasks, lists, and folders you ever created

Click Bulk Actions to perform an action against multiple tasks at once (i.e. discard them, move them to a different list).
